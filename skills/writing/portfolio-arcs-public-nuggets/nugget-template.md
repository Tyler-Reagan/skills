# Public nugget template

Flat file: `public-portfolio-nuggets/{id}.md`

Hard body budget. Stack is **texture** only (frontmatter list). No resume fields.
No Spine. No site/component instructions.

```markdown
---
id: {kebab-case-unique}
op: create | update          # last write disposition (informational)
source: internal/…           # suite or collection path this was distilled from
employer: {Company or omit}
role: Software engineer
timeline: {e.g. Aug 2025 – Jul 2026}
stack:
  - Next.js
  - TypeScript
  - …
---

# {Short product-story title}

**Thesis:** {One line — transferable SWE skill, not epic name.}

## Problem
{≤3 sentences. Public-safe constraints only.}

## Drove
- {Bullet 1 — agency}
- {Bullet 2}
- {Bullet 3}

## Judgment
- {≤3 bullets — decisions/trade-offs a skeptic can probe}
- …
- …

## Outcome
{1–2 sentences. Qualitative only — never invent metrics.}
```

## Budget bar

| Section | Cap |
| --- | --- |
| Thesis | 1 line |
| Problem | ≤3 sentences |
| Drove | exactly 3 bullets preferred (2–3 OK) |
| Judgment | ≤3 bullets |
| Outcome | 1–2 sentences |
| stack | 4–8 names |
